5. Group of selectors

A comma-separated list of selectors represents the union of all the elements selected by each of the individual selector in the list.(A comma is U+002C ) For example, in CSS when several selector share the same declarations, they may be grouped into a comma-separated list. White space may appear before and/or after the comma

CSS example:

In this example, we condense three rules with identical declarations into once. Thus ,

h1, { font-family: sans-serif }
h2, { font-family: sans-serif }
h3, { font-family: sans-serif }

is quivalent to:

h1, h2, h3 { font-family: sans-serif }
